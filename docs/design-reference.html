Container With Most Water
Medium · 20 Two Pointers time 1000ms mem 128mb

Given n non-negative heights, pick two lines that with the x-axis form a container holding the most water. Return that maximum area.

Input

First line: integer n. Second line: n space-separated heights.

Output

A single integer — the maximum area.

Constraints

2 ≤ n ≤ 1e5 0 ≤ height[i] ≤ 1e4

// two pointers from both ends
int maxArea(vector<int>& h) {
    int l = 0, r = h.size() - 1, best = 0;
    while (l < r) {
        int area = min(h[l], h[r]) * (r - l);
        best = max(best, area);
        h[l] < h[r] ? ++l : --r;
    }
    return best;
}
